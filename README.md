# Basketball projekt (NBA API)

## Opis projekta

Projekt omogoča zajem, analizo in grafični prikaz podatkov o NBA igralcih.

Uporabnik lahko:
- poišče igralca,
- pridobi njegove osnovne podatke,
- pridobi statistiko tekem (game log),
- primerja dva igralca,
- pregleda seznam ekip,
- prikaže podatke z grafi (stolpični in tortni diagram).

Projekt uporablja knjižnico `nba_api`, zato ne uporablja neposrednega spletnega scrape-anja.

## Zahteve za delovanje

Za zagon programa je potrebno imeti nameščene naslednje knjižnice:
- nba_api
- pandas
- requests
- matplotlib

Namestitev vseh knjižnic:

```bash
pip install -r requirements.txt
```
